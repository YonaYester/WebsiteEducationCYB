TwoFish Encryption


TwoFish Encryption is a symmetric key block cipher. Uses the same key on both sides and always operating compounding blocks of data, usually 128 bits or more. Twofish uses a structure called a Feistel network, which splits data into two halves and processes them with rounds of mixing, substituting, and combining. It uses:A 16-byte block size (128 bits of data), a variable-length key (128, 192, or 256 bits), and 16 rounds of operations to mix the data thoroughly.

The Basics

  1. A secret key and plaintext are inputed.
  2. The plaintext is split into 32-bit words. Split the data into a left half and a right half. This helps the algorithm mix the message in a secure way.
  3. Mix the key using key schedule (A process that turns the main key into many small pieces (subkeys), one for each round.)
  4. Pre-whiteningis when it combines your message with a piece of the key using something called XOR (think of this like scrambling with a secret sauce).
  5. Does 16 rounds of mixing, scrambles the message using math and the mini-keys. It swaps and mixes the two halves of the message again and again.
  6. Finally post-whitening, one final layer of scrambling with part of the key.
